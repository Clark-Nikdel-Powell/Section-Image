Retrieves a media object by term.

== Description ==

This plugin adds a Media Category taxonomy to Media items and allows a media object to be selected by a term.

== Installation ==

1. Upload the `section-image` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Place `get_section_image( 'term' )` in your templates.
